# Inventory API Server

## Summary

Storing items for an eCommerce API.

## Installation

Import schema.sql for Database Table 'items'

`npm install`  
`npm fund`  
`npm start`

## Routes

**GET** `/items` - Get all items  
**POST** `/items` - Add an item  
**PUT** `/items/1` - Update an item by ID  
**DELETE** `/items/1` - Delete an item by ID
